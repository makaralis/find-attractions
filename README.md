 # welcome to find-attractions web app
 
 Check-out the deployment link: https://find-attractions.netlify.app/


 Technologies I used:

For Front-End:
- React.js
- Recoil
- Hooks
- Styled components
- Deployed to netlify

For Back-End:
- Node with Express
- MongoDB with mongoose
- Deployed to Heroku
